# Website Foodstore Gwenny

## Project
Als proejct heb ik gekozen om een website te maken voor mijn moeder. Hiervoor heb ik gekozen omdat zij een Vishandel-Traiteur geeft en al bijna een halfjaar uitgebreid heeft naar een automatenshop. Voor Vishandel-Traiteur Jonas bestaat er al een website die toch nog regelmatig gebruikt wordt. Dus zou het ook handig zijn om voor de automatenshop een website te hebben. Een eventuele uitbreiding zou kunnen zijn dat de website van Vishandel-Traiteur Jonas vernieuwt wordt en dat mensen kunnen bestellen via de website.
